How it works?

Select the value of the (#)Hashtag. You can take any popular hashtag value for example #influencer, #competetion, #influencermaketing, etc.

Enter a valid integer to limit the number of records.

Note: In case the total tweets are less than the mentioned number to limit the records, then all the tweets will be fetched irrespective of entered value to limit the records.
